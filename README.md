## Requirements

# **Creation Kit is a hard requirement - Make sure you have this first!**

## REQUIRED GAME VERSION - Anniversary Edition - 1.11.221.0

### This mod list works best if installed on a SSD.

### It will require 240 GB of space (the downloads can be deleted once it's working to save space 74 GB).

### You need a clean & legal copy of Fallout 4 from Steam, as well as all DLC, EXCEPT the High Resolution Texture Pack.

### ⚠️ Do NOT use the downgrader. This list requires the Anniversary Edition version of the game.

## Install Steps - For Nexus downloaded file

#### Step 1 - Folder set-up (Optional Step)

*(This step is optional if you want to keep your file structure clean and don't have your own method)*

Create a folder called "Modding" anywhere on your SSD, away from protected folders like Program Files

In this, create a folder called "Wasteland Genesis"

And create a folder called "Wabbajack"

And create a folder called "Nexus Files".

My file structure looks like this

C:Modding

C:Modding\Nexus Files

C:Modding\Wabbajack

C:\Modding\Wasteland Genesis

![image37](https://github.com/user-attachments/assets/7605916c-e20f-4d23-ac11-2e332447ccb7)

#### Step 2 - Downloading Wasteland Genesis

**\*\* From Nexus \*\***

Download Wasteland Genesis.zip

*(You can put this into your C:Modding\Nexus Files folder if you use the same structure as me)*

Unzip Wasteland Genesis.zip and save for later

The contents of your folder should look like this:

![image23](https://github.com/user-attachments/assets/243744e1-4756-430c-a466-2427bfa841d4)

#### Step 3 - Download Wabbajack

You will also need to download and install Wabbajack from here <https://www.wabbajack.org/>

Put the Wabbajack.exe into your Wabbajack folder

*(Note: I've already run Wabbajack.exe which is why I have the extra contents in this pic)*

![image9](https://github.com/user-attachments/assets/be196543-8139-455e-af23-ab279d9c77b6)

#### Step 4 - Run Wabbajack

Open the Wabbajack file and select the Wabbajack.exe

![image28](https://github.com/user-attachments/assets/a7c4f9c4-20e0-4740-8d06-c1715635719c)

#### Step 5 - Using Wabbajack

From the Welcome page select "Get Started" - The big pink box at the bottom of the page!

[![image30](https://raw.githubusercontent.com/FirebirdGaming/Images/main/Wabbajack%20launcher.jpg)](https://raw.githubusercontent.com/FirebirdGaming/Images/main/Wabbajack%20launcher.jpg)

This will take you to the next screen where you can select the install from disk option, which can be found in the top right of the screen

[![image50](https://raw.githubusercontent.com/FirebirdGaming/Images/main/Wabbajack%20Install%20from%20file.jpg)](https://raw.githubusercontent.com/FirebirdGaming/Images/main/Wabbajack%20Install%20from%20file.jpg)

Under Target Modlist select the file "Wasteland Genesis.wabbajack" (not the .meta.json file) from the Nexus Folder that you created earlier.

![image32](https://github.com/user-attachments/assets/d1e805c9-de4a-4fee-bdb3-67f588d3295f)

On the next screen choose your file location on the left of the screen

[![image51](https://raw.githubusercontent.com/FirebirdGaming/Images/main/Wabbajack%20File%20Location.jpg)](https://raw.githubusercontent.com/FirebirdGaming/Images/main/Wabbajack%20File%20Location.jpg)

The other 2 fields fill themselves out, but you can change them if you want

Then hit the install button

## Launching Wasteland Genesis

#### Step 9 - Run Wasteland Genesis

Click on install folder and it will take you to the folder with all the mods, MO2 and other files needed to run this mod list

![image10](https://github.com/user-attachments/assets/18119724-c262-448e-a001-69935ac90008)

#### Step 10 - Run Mod Organizer

Now select Wasteland Genesis.exe. The very last file which will open a portable version of MO2

#### Step 11 - Run the game!

Click Run on the F4SE Loader on the right of the screen and the game should load.

![image31](https://github.com/user-attachments/assets/f13ee377-09fa-49f8-a2f9-d327458e206e)
